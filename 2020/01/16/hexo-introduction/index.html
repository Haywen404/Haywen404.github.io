安装

首先要在cmd里面进到你想要安装的文件夹下（里面），然后再在这个文件夹下执行Hexo和npm安装的相关命令。当然这些的前提条件是你要提前安装好Git和Node.js，如果你没有安装，可以分别点击下面两个链接查看安装教程。

Git
Node.js

进入到指定文件夹下

进入cmd

  • Windows键(有四个小方形那个) + R
  • 输入”cmd”,回车

我是安装在C盘的Blog文件夹下，所以我接下来的命令就是：

  • cd ..(向上走一级，多次重复一级一级的往上走走到C盘根目录)
    或者是
  • cd C:(直接来到了C盘根目录)
    然后是进入指定文件夹
  • cd Blog(先在文件资源管理器(我的电脑)里面的C盘先建好Blog文件夹，然后直接进入)
    或者先
  • md Blog(创建Blog文件夹（注意：这里创建的Blog在我的电脑中显示的是blog）)
    然后
  • cd Blog(进入文件夹)

进入到指定文件夹之后就可以运行Hexo和npm的相关安装命令了。
